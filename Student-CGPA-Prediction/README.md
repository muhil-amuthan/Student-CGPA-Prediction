# Student CGPA Prediction

A small example project that generates synthetic student performance data and uses a linear regression model to predict CGPA.

## Files

- `student_cgpa_prediction.py`: Python script that creates synthetic data, trains a linear regression model, evaluates it, and saves a prediction plot.
- `requirements.txt`: List of Python dependencies.
- `output.png`: Visual output file showing actual vs predicted CGPA.

## Setup

1. Create and activate a virtual environment.
2. Install dependencies:

```bash
pip install -r requirements.txt
```

## Run

```bash
python student_cgpa_prediction.py
```

The script will generate synthetic data, train the model, print evaluation metrics, and save `output.png`.
