# Student CGPA Prediction using Linear Regression

## About the Project

This project predicts a student's Final CGPA using Machine Learning.

The prediction is based on:
- Attendance
- Internal Marks
- Aptitude Score
- Programming Score

I used the Linear Regression algorithm from Scikit-Learn to build the model.

## Technologies Used

- Python
- Pandas
- Scikit-Learn

## How It Works

1. Load the student dataset.
2. Split the data into training and testing sets.
3. Train a Linear Regression model.
4. Predict the CGPA for test students.
5. Evaluate the model using Mean Squared Error (MSE).
